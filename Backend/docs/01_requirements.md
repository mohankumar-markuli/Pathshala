# Pathshala — Institution Management Platform

> **Type:** Backend REST API &nbsp;|&nbsp; **Stack:** Node.js, Express.js &nbsp;|&nbsp; **Database:** MongoDB &nbsp;|&nbsp; **Model:** White-Label SaaS

## 1. Overview

Pathshala is a white-label backend REST API platform that any private institution can deploy under their own branding. It digitizes core academic and administrative operations — enabling Admins, Teachers, Students, and Parents to interact through a single, secure system.

## 2. Objectives

- Role-based access control for all user types.
- Automate attendance, grading, timetable, and fee workflows.
- Secure data handling via JWT authentication.

## 3. Modules in Scope

| #  | Module                  | Purpose                                              |
|----|-------------------------|------------------------------------------------------|
| 1  | User Management         | Registration, login, role assignment                 |
| 2  | Student Management      | Enrollment, profiles, leave requests and approvals   |
| 3  | Staff Management        | Teacher & non-teaching staff records                 |
| 4  | Class & Section         | Class creation, sections, timetable                  |
| 5  | Attendance              | Daily tracking for students and staff                |
| 6  | Examination & Grades    | Exam scheduling, marks, report cards                 |
| 7  | Fee Management          | Fee structure, payments, receipts                    |
| 8  | Announcements           | School-wide notices and communication                |

## 4. Technical Stack

| Layer          | Technology              |
|----------------|-------------------------|
| Runtime        | Node.js                 |
| Framework      | Express.js              |
| Database       | MongoDB               |
| Auth           | JWT (JSON Web Tokens)   |
